Generate Purchase Order from Sale Order

This module give feature to create purchase Order / RFQ request to your supplier/vendor for Product you want from them based on sales request(Only if not in Done or Cancel State) from your customer.

Key Features:

  • You can create purchase order directly from sale order.
  • Allow product quantity and price modification on order generation .
  • Generate multiple purchase order of the customer easily with one-click wizard based on Sale Order only in Draft & Done State .

How To Use


Create Sale Order

After installing this module when you create Sale Order the "Create Purchase Order"button will appear in sale quotation.


Convert Purchase Order

When you click on "Create Purchase Order" button the new wizard will open with current purchase order line data.


You can see that New Order is created as Purchase Order.
